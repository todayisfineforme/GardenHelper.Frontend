# GardenHelper Frontend 

Garden Helper is a React app that allows users to plan and store their ideal gardens. utilizes the Trefle API to provide plant information.

## Installation
clone repo then 

```bash
npm install
```
to install dependencies

## Usage

```node.js
npm start
```
